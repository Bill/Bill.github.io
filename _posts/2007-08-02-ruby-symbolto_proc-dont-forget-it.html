This is your code:

puts %w{A B}.collect{|e| e.downcase}.inspect

This is my cooler code:

puts %w{A B}.collect( &:downcase).inspect

This is how the Ruby Extension Project makes it work:

class Symbol
  def to_proc
    proc { |obj, *args| obj.send(self, *args) }
  end
end
